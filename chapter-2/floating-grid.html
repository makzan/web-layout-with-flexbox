Floating grid

Revisiting the grid layout with float.

Row column grid system.

$columns-count: 12;
@media screen {
  @for $i from 1 through $columns-count {
    .col-#{$i} {
      width: 100%/$columns-count * $i;
    }
  }
}

Applying mobile first approach

$columns-count: 12;
$breakpoint: 800px;
@media screen {
  @for $i from 1 through $columns-count {
    .small-#{$i} {
      width: 100%/$columns-count * $i;
    }
  }
}
@media screen and (min-width: $breakpoint) {
  @for $i from 1 through $columns-count {
    .medium-#{$i} {
      width: 100%/$columns-count * $i;
    }
  }
}
